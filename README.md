# REQUIREMENTS
typescript version: 4.9.5
react version: 19.2.7
node version: 16.18.126

# SETUP
- Create .env file inside the root folder and add the following:
  ```
  REACT_APP_API_URL=http://localhost:8080
  ```

# RUNNING
after [backend](https://github.com/BernardWasTaken/demo-backend) is running locally, run ```npm install && npm run start```
